# Free Color Conversion API

Free color conversion API - HEX/RGB/HSL/CMYK/OKLCH plus nearest CSS name and closest Pantone-style match.

**Live tool & docs:** https://color.wrapper-agency.com

## Quickstart - free API, no key required

```bash
curl "https://color.wrapper-agency.com/api/v1/color?value=%231DB954"
```

Clean JSON, open CORS, no signup. Try the interactive tool and full docs at https://color.wrapper-agency.com
